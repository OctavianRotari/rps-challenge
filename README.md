## To play the game

0. git clone https://github.com/OctavianRotari/rps-challenge.git 

## Usage
0. run `bundle` to install the required gems (assuming Bundler is already installed).
0. run `rackup`
0. point your browser to http://localhost:9292
0. play!

## Running tests
